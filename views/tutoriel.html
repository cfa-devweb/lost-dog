Tutoriel pour l'utilisation de l'application mobile

Pour consulter les animaux perdus et trouvés

1. Quand vous ouvrez votre application, cliquer sur 'Album photo'

2. Si vous voulez consulter les animaux trouvés, taper sur 'trouvé', les animaux perdu, allez sur 'perdu'. Et enfin, si vous voulez en adopter, choisissez 'A adopter'.

3.Choisir le type de l'animal que vous recherchez, en cliquant soit sur chien, soit sur chat

4. Vous pouvez choisir le moment où l'annnonce a été publiée

5. Vous avez les annonces qui s'affichent une par une. Vous avez la possibilité de zapper en cliquant sur les flêches du bas

Pour ajouter un animal (perdu, trouvé ou a adopté)

1. Sur l'accueil cliquer sur 'Ajouter un animal'. Ensuite selon la situation qui vous convient cliquer sur "Mon album photo". Dans le cas d'un animal trouvé il est possible de prendre une photo avec GPS en cliquant sur "Prendre photo (GPS)"

2.Lorsque votre écran s'affiche comme l'image ci-contre, taper sur le type d'animal que vous ajouter.

3.Enfin, rédiger votre publication en mettant le plus d'information possible concernant l'animal.
